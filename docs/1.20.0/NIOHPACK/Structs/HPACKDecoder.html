HPACKDecoder

public struct HPACKDecoder

An HPACKDecoder maintains its own dynamic header table and uses that to decode indexed HTTP headers, along with Huffman-encoded strings and HPACK-encoded integers.

There are two pieces of shared state: the dynamic header table and an internal ByteBuffer used for decode operations. Each decode operation will update the header table as described in RFC 7541, appending and evicting items as described there.

Note

This type is not thread-safe. It is designed to be owned and operated by a single HTTP/2 stream, operating on a single NIO EventLoop.
