FailedToAddIndexedHeader

public struct FailedToAddIndexedHeader<Name, Value> : NIOHPACKError where Name : Collection, Value : Collection, Name.Element == UInt8, Value.Element == UInt8

A new header could not be added to the dynamic table. Usually this means the header itself is larger than the current dynamic table size.
