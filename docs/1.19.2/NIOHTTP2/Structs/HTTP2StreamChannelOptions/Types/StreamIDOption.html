StreamIDOption

public struct StreamIDOption : ChannelOption

StreamIDOption allows users to query the stream ID for a given HTTP2StreamChannel.

On active HTTP2StreamChannels, it is possible that a channel handler or user may need to know which stream ID the channel owns. This channel option allows that query. Please note that this channel option is get-only: that is, it cannot be used with setOption. The stream ID for a given HTTP2StreamChannel is immutable.

If a channel is not active, the stream ID will not be present, and attempting to get this channel option will fail.
